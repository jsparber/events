Thanks for submitting your event!

Someone from the community will review your event and publish it as soon as possible in the events website

In the meantime, you can:

  • Read our code of conduct
  • See all the current events
  • Read more about Open Source Design

What actually happens (gory details)

Submitting a event sends a pull request to the Open Source Design events repository. Someone needs to merge the pull request. Merging triggers the publication to the Open Source design events website
